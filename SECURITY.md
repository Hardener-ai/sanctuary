# Security Policy

Sanctuary by Hardener is a security product. We take vulnerability reports
seriously. The project lives under the Hardener-ai GitHub organization.

## Reporting a vulnerability

If you discover a security issue in Sanctuary, please report it privately. Do
not file a public GitHub issue for security vulnerabilities.

Email: hello@hardener.ai

We commit to:
- Acknowledging receipt within 48 hours
- Providing an initial assessment within 7 days
- Coordinating disclosure with the reporter

## Bug bounty

We do not currently offer paid bounties. We may launch a paid program in v0.2
once Endpoint Security entitlement-based enforcement ships.

## Supported versions

Currently we support only the latest v0.1 release. Older releases receive no
security updates.
